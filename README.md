# ✨ Overview

This repository contains my personal developer portfolio.

The website was designed and built from scratch to represent both my technical skills and my personality. Rather than using a traditional portfolio template, I wanted to create an immersive experience with interactive 3D graphics, smooth animations, and a premium user interface.

The portfolio highlights my projects, technical skills, experience, and interests while maintaining excellent performance and responsiveness across devices.

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/portfolio-chandrathanaya.git
```

## Navigate into the project

```bash
cd portfolio-chandrathanaya
```

## Install dependencies

```bash
npm install
```

## Start development server

```bash
npm run dev
```

Open

```
http://localhost:5173
```
